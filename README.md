Este ejercicio es una simulación de una tienda online elavorada utilizando los principios de programación orientada a objetos(POO).
Utilizando la estructura de clases (class) hemos integrado funciones que permiten gestionar el inventario de la tienda online, como:
· Agregar productos al inventario.
· Ver el inventario y calcular el valor total que hay en el.
· Eliminar productos y buscarlos.
· Actualizar el stock de tu tienda.

El objetivo del ejercicio es demostrar la creacion de clases y la aplicación de los conceptos y métodos básiscos de programacion con Python.

El sistema simula la gestión del inventario de la tienda online.
